### 2.流水线技术

* 15年之前考得比较多,16年考过一次

* 流水线周期:各子任务中执行时间最长的(最慢的)子任务的执行时间

  ```
  1.取指令   2,分析执行,  3执行指令  (分别花1ns,  2ns,  3ns),
  执行100次
  串行的时候100x(1+2+3)=600ns
  
  tn=执行一条指令需要的时间+(n-1)* 流水线周期
  
  流水线周期
  tn=(1+2+3)+(100-1)*3=303ns
  ```
